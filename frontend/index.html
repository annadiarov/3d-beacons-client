Background

Recent progress in structure prediction and generative protein design has made it possible to explore protein interactions at proteome scale. Protein structure resources such as the AlphaFold Database provide high-quality predicted structures for many proteins, but most design workflows still need domain-level inputs and systematic downstream processing before binder design can be attempted at scale.

For Bindome, protein structures from the AlphaFold Database are processed into domain-level targets. These domains are then used as input for BindCraft to design candidate binders against each domain. The resulting models are organized by UniProt accession, domain, and design run, and are exposed here as searchable complex models with associated coordinate files and metadata.

This first website prototype focuses on making these models easy to search, inspect, and download. The scientific description can be expanded here as the release text and manuscript mature.

About this release

This release provides an initial searchable collection of BindCraft-designed protein binder models and their predicted complexes. Entries are queried by UniProt accession and grouped by target/model identifier. Each entry links to a coordinate file that can be viewed in Mol* or downloaded for external analysis.

The current release should be considered a structural resource and design catalogue. Users should evaluate individual designs according to their own downstream criteria before experimental use.

What’s next?

Planned server updates include richer search filters, improved model detail pages, weekly database statistics, downloadable summary reports, archive checksums, and clearer links between designed binders, target domains, and known biological annotations.

The first public version focuses on the human proteome. Future releases are planned to expand Bindome to additional proteomes and additional biologically relevant subsets, including binders that may interfere with known protein-protein interactions or known post-translational modification sites.
